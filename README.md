# Sales Data ETL Pipeline

This project demonstrates a simple data engineering workflow using Python, SQL, Pandas, and PySpark.

Pipeline Steps:

1. Raw sales dataset ingestion
2. Data cleaning and preprocessing using Pandas
3. Loading processed data into SQL database
4. Querying structured data using SQL
5. Large-scale data aggregation using PySpark

Tech Stack:
- Python
- Pandas
- SQL (SQLite)
- PySpark

Key Features:
- Data cleaning
- ETL workflow
- SQL aggregation
- Distributed processing with PySpark
